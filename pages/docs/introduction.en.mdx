# Introduction

Scratch Auth is a simple OAuth service for Scratch. It provides an easy-to-understand API for developers and a smooth login experience for users. By using Scratch Auth, you can easily implement OAuth functionality on your website.

## Related Service Status

This package uses the APIs of external services, so it may be affected by the status of these related services.

- [Scratch Auth](https://stats.uptimerobot.com/4Ggz4Fzo2O)
- [PenguinMOD](https://status.penguinmod.com)
