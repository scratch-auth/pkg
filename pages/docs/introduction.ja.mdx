# はじめに

Scratch Auth はScratch用のシンプルなOAuthサービスです。開発者にはわかりやすいAPIを、ユーザーにはスムーズなログイン体験を提供します。Scratch Auth を使用することで、簡単にOAuth機能をサイトに実装することができます。

## 関連サービスステータス

このパッケージは、外部サービスのAPIなどを使用している機能があります。その為関連サービスのステータス状況の影響を受けます。

- [Scratch Auth](https://stats.uptimerobot.com/4Ggz4Fzo2O)
- [ペンギンMOD](https://status.penguinmod.com)
